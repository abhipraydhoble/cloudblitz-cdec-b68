## Command Mode (default)

| Key | Action |
|----|-------|
| `yy` | copy line |
| `cc` | cut line |
| `dd` | delete line |
| `p`  | paste |
| `dw` | delete word |
| `yw` | copy word |
| `cw` | cut word |
| `u`  | undo |
| `H`  | cursor at top |
| `L`  | cursor at bottom |

---

## Insert Mode

| Key | Action |
|----|-------|
| `i` | insert text |
| `o` | new line below |
| `O` | new line above |

---

## Execution Mode (Command-Line)

| Command | Action |
|--------|-------|
| `:w` | save |
| `:q` | quit |
| `:wq` / `:x` | save and quit |
| `:q!` | force quit |
| `:set nu` | show line numbers |
| `:set nonu` | hide line numbers |
| `:%s/old/new` | replace word globally |

---

## Visual Mode

| Key | Action |
|----|-------|
| `v` | select text |
| `V` | select line |
| `y` | copy |
| `d` | delete |
| `c` | cut |
| `p` | paste |
